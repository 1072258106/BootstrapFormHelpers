Countries bootstrap-countries.js

About countries

For adding a drop-down of countries or display country name from country code, include bootstrap-countries.js once alongside the other JS files.

Use cases

Simple example for loading the list of countries

<select class="input-medium countries"></select>

Example for loading the list of countries and selecting a default country

<select class="input-medium countries" data-country="US"></select>

Example for loading the list of countries in JavaScript

<button onclick="$('#countries1').countries()" class="btn">Load Countries</button>
<select id="countries1" class="input-medium"></select>

Example for loading the list of countries in JavaScript and selecting a default country

<button onclick="$('#countries2').countries({country: 'US'})" class="btn">Load Countries</button>
<select id="countries2" class="input-medium"></select>

Example for displaying the country name from a country code

<span class="countries" data-country="US"></span>

Example for displaying the country name from a country code in JavaScript

<button onclick="$('#countries3').countries({country: 'US'})" class="btn">Show Country</button>
<span id="countries3"></span>

States bootstrap-states.js

About states

For adding a drop-down of states of a country or display state name from state and country code, include bootstrap-states.js once alongside the other JS files.

Use cases

Simple example for loading the list of states for a country

<select class="input-medium states" data-country="US"></select>

Simple example for loading the list of states for a country and selecting a default state

<select class="input-medium states" data-country="US" data-state="CA"></select>

Example for loading the list of states for a country in JavaScript

<button onclick="$('#states1').states({country: 'US'})" class="btn">Load States</button>
<select id="states1" class="input-medium"></select>

Example for loading the list of states for a country and selecting a default state in JavaScript

<button onclick="$('#states2').states({country: 'US', state: 'CA'})" class="btn">Load States</button>
<select id="states2" class="input-medium"></select>

Example for loading the list of states based on a country selector

<select id="countries_states" class="input-medium countries" data-country="US"></select>
<select class="input-medium states" data-country="countries_states"></select>

Example for displaying the state name from a state and country code

<span class="states" data-country="US" data-state="CA"></span>

Example for displaying the state name from a state and country code in JavaScript

<button onclick="$('#states3').states({country: 'US', state: 'CA'})" class="btn">Show State</button>
<span id="states3"></span>

Phone bootstrap-phone.js

About phone

For simple formatting of phone numbers, include bootstrap-phone.js once alongside the other JS files.

Use cases

Simple example for formatting a phone number in input text

<input type="text" class="input-medium phone" data-format="+1 (xxx) xxx-xxxx">

Simple example for displaying a formatted phone number

<span class="phone" data-format="+1 (xxx) xxx-xxxx" data-number="5555555555"></span>

Example for formatting a phone number based on a country

<input type="text" class="input-medium phone" data-country="US">

Example for formatting a phone number based on a country selector

<select id="countries_phone" class="input-medium countries" data-country="US"></select>
<select class="input-medium phone" data-country="countries_phone"></select>

Example for getting the unformatted phone number

<input id="phone1" type="text" class="input-medium phone" data-format="+1 (xxx) xxx-xxxx">
<button onclick="alert($('#phone1').data('number'));return false;" class="btn">Get Phone</button>
